# 学习Golang的Gin框架的Demo

## 安装

- Clone this project
    ```bash
    $ git clone  git@github.com:linganmin/learn-gin.git
    ```

- Install `kardianos/govendor`
    ```bash
    $ go get github.com/kardianos/govendor
    ```

- Install `codegangsta/gin`
    ```bash
    $ go get github.com/codegangsta/gin
    ```

## 启动

```bash

cd learn-gin

# 使用热加载启动
gin run main.go
```
